# ResumeAI

> Land your dream job with an AI-crafted resume

ResumeAI uses advanced AI to build tailored, ATS-optimized resumes in minutes — not hours. Paste a job description and watch your resume transform to match exactly what hiring managers want.

Generated by **[Skylia](https://skylia.dev)** AI Startup Builder.

## Stack

| Layer | Service |
|-------|---------|
| Framework | Next.js 14 + TypeScript |
| Styling | Tailwind CSS + shadcn/ui |
| Hosting | Vercel |
| Database | Neon + Prisma |
| Auth | Clerk |
| Payments | Stripe |
| Email | Resend |
| AI | OpenAI |


## Getting started

```bash
npm install
cp .env.example .env.local
# Fill in your environment variables
npm run dev
```

### Where to get them

- `DATABASE_URL` — Neon PostgreSQL connection string (Neon dashboard)
- `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY` — Clerk publishable key for frontend auth (Clerk dashboard)
- `CLERK_SECRET_KEY` — Clerk secret key for server-side auth (Clerk dashboard)
- `CLERK_WEBHOOK_SECRET` — Webhook secret to verify Clerk user sync events (Clerk dashboard)
- `STRIPE_SECRET_KEY` — Stripe secret key for payment processing (Stripe dashboard)
- `NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY` — Stripe publishable key for frontend Stripe.js (Stripe dashboard)
- `STRIPE_WEBHOOK_SECRET` — Webhook secret to verify Stripe billing events (Stripe dashboard)
- `STRIPE_PRO_PRICE_ID` — Stripe price ID for the Pro monthly plan (Stripe dashboard)
- `STRIPE_CAREER_PRICE_ID` — Stripe price ID for the Career monthly plan (Stripe dashboard)
- `OPENAI_API_KEY` — OpenAI API key for all AI resume features (OpenAI platform)
- `RESEND_API_KEY` — Resend API key for transactional emails (Resend dashboard)
- `NEXT_PUBLIC_POSTHOG_KEY` — PostHog project API key for analytics (PostHog dashboard)
- `NEXT_PUBLIC_POSTHOG_HOST` — PostHog ingestion host URL (PostHog dashboard)
- `SENTRY_DSN` — Sentry DSN for error monitoring (Sentry dashboard)
- `NEXT_PUBLIC_APP_URL` — Public base URL of the deployed application (Vercel environment)
